# 山丘

## 介绍

这里是「山丘代码铺」的文章归档仓库。

我是一名大三后端实习生，正在从 Java 后端走向 AI Agent、后端工程和秋招准备。这个仓库主要记录我在实习、学习和项目实践中的思考与总结。

内容不追求一上来就讲得很“高大上”，而是尽量把一个个原本看不懂、听不明白的技术点，拆成普通人也能理解的过程。

> 山丘不急，慢慢往上爬。

## 内容方向

目前主要会更新这些内容：

* AI Agent 学习笔记
* 后端工程实践总结
* 实习项目复盘
* 秋招八股与面试准备
* Java / Go / Python 学习记录
* OAuth、HMAC、JWT、WebSocket、MQTT 等工程知识
* LiveKit、RAG、MCP、Agent 框架等 AI 工程内容
